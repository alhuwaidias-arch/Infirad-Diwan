# How to Enable GitHub Pages for Diwan-Al-Maarifa

## Quick Steps to Get Your Live Website

Follow these simple steps to enable GitHub Pages and get your website live at:
**https://alhuwaidias-arch.github.io/Diwan-Al-Maarifa/**

---

## Step-by-Step Instructions

### 1. Go to Repository Settings
- Navigate to: https://github.com/alhuwaidias-arch/Diwan-Al-Maarifa
- Click on **Settings** tab (top right)

### 2. Navigate to Pages Section
- In the left sidebar, scroll down and click on **Pages**
- Or go directly to: https://github.com/alhuwaidias-arch/Diwan-Al-Maarifa/settings/pages

### 3. Configure GitHub Pages
Under "Build and deployment":
- **Source**: Select "Deploy from a branch"
- **Branch**: Select "main"
- **Folder**: Select "/ (root)"
- Click **Save**

### 4. Wait for Deployment
- GitHub will automatically build and deploy your site
- This usually takes 1-2 minutes
- You'll see a message: "Your site is live at https://alhuwaidias-arch.github.io/Diwan-Al-Maarifa/"

---

## Troubleshooting

### If the site doesn't load:
1. Wait 2-3 minutes for initial deployment
2. Check the Actions tab to see deployment status
3. Clear your browser cache (Ctrl+F5)

### If you see a 404 error:
1. Make sure `index.html` exists in the root directory (it does ✅)
2. Check that the branch is set to "main"
3. Verify the folder is set to "/ (root)"
